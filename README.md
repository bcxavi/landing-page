# Landing Page – The Odin Project

## Description

This project is a responsive landing page built as part of *The Odin Project* curriculum. It recreates a complete webpage layout using HTML and CSS, focusing on structure, layout, and styling with modern techniques like Flexbox.

The page includes multiple sections such as a hero, information cards, testimonial (quote), call-to-action, and footer.

---

## Features

* Semantic HTML structure (`header`, `main`, `section`, `footer`)
* Layout built using **Flexbox**
* Consistent spacing and alignment across sections
* Reusable button styles with variations
* Custom typography using **Roboto (Google Fonts)**
* Structured and maintainable CSS

---

## Sections Included

* **Header** – Navigation and logo
* **Hero** – Main message with image and call-to-action
* **Information Section** – Grid of cards with images and text
* **Quote Section** – Testimonial-style content
* **Call to Action (CTA)** – Highlighted signup section
* **Footer** – Copyright information
